apply(skin) → {boolean}

Applica una skin rimuovendo tutte le classi skin precedenti e aggiungendo quella nuova. Salva la preferenza in localStorage. La skin viene applicata tramite classe CSS: body-skin-{nome} che attiva le CSS custom properties definite in styles-skin.css
Parameters:
Name Type Description
skin string Nome della skin da applicare ("red", "blue", "green")
Source:
  • skin.js, line 114
Returns:
True se la skin è stata applicata, false se invalida
Type
boolean
Examples
skinManager.apply("red"); // Applica skin rossa
if (skinManager.apply("invalid")) {
  console.log("Skin valida");
} else {
  console.log("Skin non esistente");
}

applyIfAllowed(skin, gameInProgress) → {SkinApplicationResult}

Tenta di applicare una skin solo se le condizioni lo permettono. Previene il cambio skin durante una partita in corso per evitare comportamenti imprevisti o distrazioni per il giocatore.
Parameters:
Name Type Description
skin string Nome della skin da applicare
gameInProgress boolean Flag che indica se il gioco è attivo
Source:
  • skin.js, line 167
Returns:
Oggetto con esito e motivo eventuale
Type
SkinApplicationResult
Examples
// Durante il gioco
const result = skinManager.applyIfAllowed("blue", true);
if (!result.ok) {
  console.log("Impossibile cambiare:", result.reason); 
  // "game_in_progress"
}
// A gioco fermo
const result = skinManager.applyIfAllowed("blue", false);
if (result.ok) {
  console.log("Skin cambiata con successo!");
}
