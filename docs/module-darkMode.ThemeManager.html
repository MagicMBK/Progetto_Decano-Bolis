Class: ThemeManager

darkMode.ThemeManager()

Gestore completo del sistema di temi light/dark

Constructor

new ThemeManager()

Crea una nuova istanza del ThemeManager. All'istanziazione carica automaticamente la preferenza salvata, crea il pulsante toggle nel DOM, e applica il tema corrente.
Properties:
Name Type Description
isDark boolean Indica se il tema dark è attualmente attivo
toggleButton HTMLButtonElement | null Riferimento al pulsante toggle
Source:
  • darkMode.js, line 44
Examples
// Inizializzazione automatica all'avvio del gioco
const themeManager = new ThemeManager();
// Carica automaticamente la preferenza salvata e crea il pulsante
// Toggle programmatico del tema
const themeManager = new ThemeManager();
themeManager.toggle(); // Passa da light a dark (o viceversa)
console.log(themeManager.getCurrentTheme()); // "dark" o "light"

toggle() → {void}

Alterna tra tema chiaro e scuro. Questa è la funzione principale chiamata quando l'utente clicca il pulsante toggle. Sequenza operazioni: 1. Inverte lo stato di isDark 2. Salva la nuova preferenza in localStorage 3. Applica il nuovo tema al documento

void
Examples
// Click sul pulsante
themeManager.toggle(); 
// Light -> Dark o Dark -> Light
// Toggle programmatico (es. da menu settings)
if (userPreference === 'dark') {
  if (!themeManager.isDark) {
    themeManager.toggle();
  }
}
